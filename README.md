Smart Task Priority Optimizer

A Python program that helps prioritize tasks based on priority, deadline and time required. It suggests an order in which tasks should be completed.

---

-> Overview

When multiple tasks are pending, it becomes difficult to decide what to do first. This program takes basic task details and generates a recommended order based on importance and urgency.

---

-> Features

- Add multiple tasks  
- Set priority (High / Medium / Low)  
- Enter deadline and time required  
- Generates a recommended task order  
- Provides simple explanation for each task  

---

-> AI Concept Used

The system uses a simple heuristic-based approach.  
Each task is given a score based on:
- priority (importance)  
- deadline (urgency)  
- time required  

Tasks are then sorted based on this score. This is similar to how AI systems use evaluation functions to make decisions.

---

-> Technologies Used

- Python  
- No external libraries
  
---

-> How to Run

1. Open the file in any Python IDE (like PyCharm)  
2. Run the program  
3. Enter task details when prompted  

---

-> Testing

- Add tasks with different priorities  
- Use different deadlines  
- Change time required  
- Observe how task order changes
